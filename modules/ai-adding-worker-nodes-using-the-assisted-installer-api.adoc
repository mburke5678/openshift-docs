= Adding worker nodes using the Assisted Installer API

[role="_abstract"]
You can add worker nodes to {sno} clusters by using the Assisted Installer REST API. 

Before you add worker nodes, you must log in to link:https://console.redhat.com/openshift/token/show[{cluster-manager}] and authenticate against the API.
