= Boot image skew enforcement

toc::[]

[role="_abstract"]
You can use boot image skew enforcement to help ensure that the boot images in a cluster are up-to-date with the {product-title} and {op-system} version being used in the cluster. Using an older boot image could cause compatibility issues within your cluster.
