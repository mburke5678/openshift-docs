= About using machine config pools to configure nodes
include::_attributes/common-attributes.adoc[]


toc::[]

By default, there are two machine config pools (MCP) created by the cluster when it is installed: `master` and `worker`.  Each MCP manages a set of nodes and applies changes made through a machine config to the appropriate nodes. 

In addition, you can apply changes to specific sets of worker nodes by using _custom machine config pools_. 

For infomration on configuring nodes by using machine config objects, see xref:../machine_configuration/machine-configs-configure.adoc#machine-configs-configure[Using machine config objects to configure nodes]
